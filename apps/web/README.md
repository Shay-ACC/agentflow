# Web App

Next.js frontend for AgentFlow.

This folder currently contains only the starter structure:
- `src/app`: App Router entrypoint
- `src/components`: future shared UI components
- `src/features`: future feature modules
- `src/lib`: future client utilities and API helpers
- `src/hooks`: future React hooks
- `src/types`: future shared frontend types
- `tests`: frontend test suite
