# API App

FastAPI backend for AgentFlow.

This folder currently contains only the starter structure:
- `app/main.py`: minimal application entrypoint
- `app/api`: future HTTP routes
- `app/core`: future configuration and integrations
- `app/models`: future SQLAlchemy models
- `app/schemas`: future request and response schemas
- `app/repositories`: future data access layer
- `app/services`: future business logic
- `app/workers`: future background jobs
- `alembic`: future database migrations
- `tests`: backend test suite
